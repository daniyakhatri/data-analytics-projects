# Operations & Vendor Analytics Dashboard (Mock Project)

## Overview
This project demonstrates the development of a Power BI dashboard designed to analyze vendor activity and operational performance across multiple service areas.

## Data
This project uses mock and simulated data created for demonstration purposes. No proprietary, confidential, or sensitive information is included.

## Key Metrics Tracked
- Vendor billing amounts
- Service area assignment
- Work status and closure times
- Cause breakdowns
- Monthly performance trends

## Data Preparation
- Cleaned and structured data from multiple sources using Python and Power Query
- Standardized dates, categories, and status fields
- Validated data before visualization to ensure accuracy

## Dashboard Features
- Interactive slicers for service area, vendor, month, and status
- Trend analysis and breakdown views
- User-focused layout design to improve usability and clarity

## Technical Challenges & Solutions
- Troubleshot data access issues to ensure consistent data refreshes and reliable dashboard reporting.
- Strengthened testing and validation workflows using Visual Studio Code before loading data into Power BI

## Tools Used
- Power BI
- Python
- Power Query
- Visual Studio Code
